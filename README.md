# 说明

## 机型
Voron v0.2 r1 120mm

从0.1升级修改，Z轴改为皮带轴

## 机械硬件
* 型材：1515氧化黑
* 结构件：ABS打印
* 线轨：林邓
* 皮带：盖茨LL-2GT-6 RF、188-2GT-6RF
* 传动轮：便宜货
* 轴承：仿NSK
* 热床铝板：6061

## 电气硬件
* 上位机：BigTreeTech CB1 v2.2 *1
* 主板：BigTreeTech Manta M8P v2.0 *1
* 打印头控制板：自制集线板 *1
* XY驱动：FYETC S2209 v3.1 *2
* Z驱动：FYETC S2209 v3.1 *1
* AB电机：雷赛 35CM04 *2
* Z电机：雷赛 35CM04 *1
* 挤出机：Orbiter v1.5 with LDO-36STH20-1004AHG *1
* 电源：MeanWell UHP-200-24 *1
* 热床硅胶加热垫：DC 60W *1
